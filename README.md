# Project 1: Rule-Based AI Chatbot (DecodeLabs)

This is the foundational phase project focusing on Control Flow, Decision-Making Logic, and AI Guardrails.

## Features Implemented:
1. **Input Sanitization:** Lowercasing, stripping whitespace, and removing punctuation.
2. **Intent Resolution:** Direct mapping of user input to specific logic gates.
3. **State & Context Tracking:** Remembering and dynamically using the user's name throughout the session.
4. **Fallback Counter & Escalation:** Automatically suggesting human support (`support@decodelabs.ai`) after 3 consecutive unrecognized inputs.
5. **New Intents Added:** Supported domains for 'about us', 'services', and 'feedback'.
